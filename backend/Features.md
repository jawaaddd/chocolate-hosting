# Ideas and Feature List

The features should not dilute the simplicity of the app. The main goal should be to create as smooth of an experience with as few roadblocks to starting a basic server you can just start play with your friends asap.

- Bells and whistles:
  - Custom subdomains
  - Basic server-icon generator
  - Curated customizeable experience integrations
    - Adventure Maps
    - Texture Packs
    - Data Packs?


## Performance

- For modern versions, default Fabric server with performance mods
- Paper for other versions where supported
